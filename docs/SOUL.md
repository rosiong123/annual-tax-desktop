## 🎯 产品定位

**年度汇算清缴智能审查及申报系统**（桌面版）

一款面向小微企业的 AI 税务合规工具，整合月报、季报、年度汇算清缴三大周期，实现**一键导入 → AI审核 → 风险预警 → 优化建议 → 申报生成**的完整闭环。

### 核心价值
- **省时**：自动化数据处理，减少手工填报
- **避坑**：AI 风险扫描，提前发现潜在问题
- **省钱**：智能税收优惠检测，最大化节税

---

## 🧠 设计理念

### 1. 三报合一
系统统一处理三种税务周期，避免企业维护多套工具：

| 周期 | 申报类型 | 核心关注点 |
|------|----------|------------|
| 月报 | 增值税 | 进项抵扣、销项开票 |
| 季报 | 企业所得税预缴 | 收入确认、成本匹配 |
| 年度汇算清缴 | 企业所得税 | 纳税调整、优惠叠加、风险排查 |

### 2. 证据链思维
税务检查的本质是**证据链审查**。系统不只是计算数字，还要验证：
- 发票是否真实有效
- 业务与发票是否匹配（三流合一）
- 调整事项是否有充分依据

### 3. 多智能体协作
采用分层 AI 架构：

```
L1 Agent（事实统一）
    ↓
L3 Agent（研产边界风险扫描）
    ↓
SLPE Agent（小微优惠路径优化）
    ↓
Final Agent（裁决结项）
```

每个 Agent 专注特定任务，通过协作完成复杂审计。

---

## 🚫 绝对不能做的事

1. **不生成虚假数据** - 系统只处理真实财务数据
2. **不替代专业判断** - AI 提供参考，最终决策在用户
3. **不存储敏感数据** - 除非明确授权，所有数据留在本地
4. **不绕过税务系统** - 申报必须通过官方渠道

## ✅ 必须遵循的约束

### 技术约束
- 使用 **Tauri + React + TypeScript** 技术栈
- 桌面端优先，支持 Windows
- 数据本地处理，隐私优先

### 业务约束
- 支持 **2023-2025** 年度数据
- 覆盖增值税（月报）、企业所得税（季报+年报）
- 兼容 Excel 导入格式

### 体验约束
- 5 步完成完整申报流程
- 关键步骤必须有进度反馈
- 错误信息要具体、可操作

## 🔑 关键概念

### 证据完整度
衡量申报材料充分性的指标（0-100分）。低于阈值时阻止申报，提示补充证据。

### 小微优惠（SLPE）
小型微利企业税收优惠。满足条件可享受 5% 实际税负率，节省大量税款。

### 研产边界
研发费用与生产成本之间的界限。边界模糊会导致加计扣除计算错误。

### 三流合一
发票、合同、资金流的一致性检查。是判断业务真实性的核心依据。
