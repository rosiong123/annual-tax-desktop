# 业务术语表

> 汇算清缴系统中使用的业务术语定义。

---

## 税务周期

| 术语 | 说明 |
|------|------|
| **月报** | 增值税及附加税费月度申报（每月 1-15 日） |
| **季报** | 企业所得税季度预缴申报（每季度終了後 15 日内） |
| **年报** | 年度企业所得税汇算清缴（次年 5 月 31 日前） |

---

## 税务概念

| 术语 | 说明 |
|------|------|
| **应纳税所得额** | 企业每一纳税年度的收入总额，减除不征税收入、免税收入、各项扣除后的余额 |
| **税率** | 企业所得税基本税率 25%，符合条件的小微利企业可享 20%，实际税负率可低至 5% |
| **小微优惠 (SLPE)** | 小型微利企业税收优惠，需同时满足：从业人数 ≤300人、资产总额 ≤5000万、应纳税所得额 ≤300万 |
| **研发费用加计扣除** | 企业研发费用可按 75%（制造业）或 100%（其他行业）加计扣除 |
| **工资薪金** | 企业支付给员工的工资、奖金、津贴等，需代扣代缴个人所得税 |

## 审计概念

| 术语 | 说明 |
|------|------|
| **证据完整度** | 衡量申报材料充分性的评分体系，低于阈值（默认 70 分）阻止申报 |
| **三流合一** | 发票流、合同流、资金流的一致性检查 |
| **体外循环** | 未通过公账的资金往来，如私户收款、现金交易 |
| **研产边界** | 研发费用与生产成本之间的界限，区分不清会导致加计扣除错误 |
| **风险评分** | 综合评估企业被查概率的分数（0-100 分） |

---

## 多智能体

| 术语 | 说明 |
|------|------|
| **L1 事实统一** | Agent-L1，负责将不同来源的财务数据统一为标准格式 |
| **L3 研产边界** | Agent-L3，负责扫描研发与生产的边界风险 |
| **SLPE 小微优惠** | Agent-SLPE，计算小微企业最优税负路径 |
| **Final 裁决结项** | Agent-Final，综合所有证据生成最终裁决 |

---

## 审计问题分类

| 分类 | 说明 |
|------|------|
| **compliance** | 合规性问题 - 违反税法的行为 |
| **human_error** | 人为错误 - 填写错误、计算错误 |
| **logic_error** | 逻辑错误 - 数据关系不合理 |
| **warning** | 风险预警 - 需要关注的异常 |

---

## 问题严重程度

| 级别 | 说明 |
|------|------|
| **critical** | 致命 - 必须立即修复，否则无法申报 |
| **high** | 高风险 - 可能导致补税或罚款 |
| **medium** | 中风险 - 需要关注，可能需要调整 |
| **low** | 低风险 - 建议优化，不影响申报 |
